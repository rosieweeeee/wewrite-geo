# 微信草稿箱 API 字段限制

> 来源：实测（2026-04-22），调用 `POST /cgi-bin/draft/add`
> 超出限制返回对应 errcode，推送失败。

## 字段长度限制（UTF-8 字节数）

| 字段 | 参数名 | 最大限制 | errcode | 建议值 |
|------|--------|---------|---------|--------|
| 标题 | `title` | 64 字节 | 45003 | ≤20个中文字（60字节） |
| 摘要 | `digest` | 120 字节 | 45004 | ≤35个中文字（105字节）或留空 `""` |
| 作者 | `author` | 8 字节（约2-4个汉字） | 45110 | 留空 `""` 或最多2个汉字 |
| 正文 | `content` | 20000 字符（约） | 45002 | HTML 控制在 15000 字符以内 |

## 其他限制

| 项目 | 限制 | 说明 |
|------|------|------|
| IP 白名单 | 必须提前在后台配置 | errcode 40164；路径：基本配置 → API IP白名单 |
| 封面图 | 必须用永久素材 media_id | 调用 `add_material` 接口上传 |
| 正文图片 | 用 `uploadimg` 接口获取 URL | 临时素材，URL 长期有效 |
| Access Token | 有效期 7200 秒 | 每次推送前重新获取即可 |

## 推荐的安全写法

```python
TITLE  = "大理·一个下午的呆"          # ≤20字
DIGEST = ""                            # 留空最安全，微信自动截取正文前120字节
AUTHOR = ""                            # 留空，避免超出2字节限制
```

## 常见 errcode 速查

| errcode | 含义 | 解决方法 |
|---------|------|---------|
| 40001 | AppID/Secret 错误 | 检查 config.yaml |
| 40164 | IP 不在白名单 | 后台 → 基本配置 → API IP白名单 添加本机 IP |
| 45002 | content 内容超限 | 精简 HTML |
| 45003 | title 超出64字节 | 缩短标题 |
| 45004 | digest 超出120字节 | 缩短摘要或留空 |
| 45110 | author 超出限制 | 缩短作者名或留空 |
| 40007 | thumb_media_id 无效 | 重新上传封面永久素材 |
| 45009 | 接口调用频率超限 | 稍后重试 |
